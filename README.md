# OpenRouteTracking
Ionic App for tracking all kinds of routes

Goals:
 - Tracking routes and viewing them live on a map
 - Saving routes for viewing later on
    - Saving locally
    - Saving on MongoDB Database
    - Saving on Google Drive (?)
    - Saving in Profiles
 - Seeing Stats on Speed, elevation gain, etc
 - Add Markers to your route after saving it for fun little notes
